# Worked Example: Bookshelve

Bookshelve is a minimal web app where users save books they want to read. It exists purely as a reference project to show how every Forge artifact connects.

This example walks through:
1. Inception outputs (Lean Canvas, Empathy Map, Trade-off Sliders)
2. Event Storming output (event-storm.yaml + CONTEXT.md)
3. One written story (four-gate review)
4. One ADR
5. One story snapshot (mid-delivery state)
6. The full Linear board state at story completion

Read it in order. Each section references the skill that produced it.

---

## The domain in one sentence

Bookshelve lets a reader save books to a reading list and mark them as read.
